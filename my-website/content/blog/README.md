# Blog posts

Create a new `.md` file in this folder and it will automatically appear on `/blog`.

Use this frontmatter format:

```md
---
title: "Post title"
date: "2026-03-29"
slug: "post-title"
excerpt: "Optional short summary"
---

Your markdown content here.
```

Images example:

```md
![Alt text](/blog-images/my-post/diagram.png)
```

Math example:

```md
Inline: $E = mc^2$

$$
\int_0^1 x^2 dx = \frac{1}{3}
$$
```
